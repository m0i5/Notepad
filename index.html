Introduction - Intermediate CSS

  • Cascading Style Sheets, meistens als CSS abgekürzt, ist eine Beschreibungssprache, die das Erscheinungsbild einer in HTML oder XML formatierten Datei (inklusive verschiedener XML-Sprachen wie SVG oder XHTML (en-US)) festlegt. CSS beschreibt, wie ein strukturiertes Element am Bildschirm, auf Papier, in Sprache oder anderen Medien dargestellt wird.

  • Kursübung: Wir haben einen neuen Ordner erstellt (external-css) mit einem neuen Sheet, dass styles.css heisst. In diesem Sheet werden alle Elemente wie z.B. body, hr, h1-h6 etc. definiert. CSS Code kann aber auch direkt in HTML geschrieben werden (internal-css), ist jedoch fehleranfällig und man muss jeden Bereich einzeln definieren.

  • Beispiel: HR's sollten zentriert als Punkte angezeigt werden.

Auch wenn man dem Kurs aufmerksam folgt, konnte man noch nicht wissen dass es Post- und Prefix increments gibt:
Postfix increment:
let x = 3;
y = x++;
// y = 3
// x = 4

let a = 2;
b = ++a;
//a = 3
//b = 3

Deshalb ist y = 4 und nicht 5!

There are a few different ways to select elements with JS:

document.getElementsByTagName("li")
gives back an array with all list items on the page, so to change the elements the code has to look like this:
document.getElementsByTagName("li")[1].style.color = "purple";

document.getElementsByClassName("btn");
also gives back an array with all classes named "btn" on the page, so to change the elements the code has to look like this:
document.getElementsByTagClass("btn")[1].style.color = "purple";

document.getElementsById("title");
also gives back the element with the defined ID. To change elements it could look like this:
document.getElementsById("title").innerHTML = "purple";

But the best selector is the querySelector()
document.querySelector() also gives back only one element, but the selectors in the braces can be combined!
For example an ID with a class("#titleBig .btn ), or a class with an element("li.item").

In this example the «eventListener» waits until the target ("button") is «clicked». When clicked, the function will be triggered.

Important: The function is written without parantheses() in this case, because the «eventListener» must wait for the click until the function should be called.
If the function is written normally like «handleClick()» it wouldn't wait for the click and the function (in this case the alert) will be instantly triggered.


So the code should run like this:
  1. searching for the button via .querySelector
  2. .addEventListener waits for «click»
  3. only after it is clicked, the function «handleClick» will be triggered
